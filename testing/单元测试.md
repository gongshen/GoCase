## 1.并行
```go
package main

import (
	"testing"
	"time"
	"os"
)

func TestA(t *testing.T)  {
	t.Parallel()
	time.Sleep(1e9)
}

func TestB(t *testing.T)  {
	if os.Args[len(os.Args)-1]=="b"{
		t.Parallel()
	}
	time.Sleep(1e9)
}

```
普通单元测试：
```go
$ go test -v
=== RUN   TestA
=== RUN   TestB
--- PASS: TestB (1.00s)
--- PASS: TestA (1.00s)
PASS
ok      test1     2.083s
```
并行单元测试：
```go
$ go test -v -args "b"
=== RUN   TestA
=== RUN   TestB
--- PASS: TestB (1.00s)
--- PASS: TestA (1.00s)
PASS
ok      test1     1.077s
```
